    Copia el archivo ZZZ_courseplay.zip a la carpeta C:\Users\[Username]\My Documents\MyGames\FarmingSimulator2013\mods. Eso es todo, amigos!

    Para instalar la versión de desarrollo, el archivo zip descargado debe ser descomprimido. El contenido de la carpeta courseplay-master en un nuevo archivo zip con el nombre de ZZZ_courseplay.zip. Después de eso, siga el procedimiento habitual (véase más arriba).
